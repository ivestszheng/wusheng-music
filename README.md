#### 介绍

本音乐网站的开发主要利用 VUE 框架开发前台和后台，后端接口用 Spring Boot + MyBatis 来实现，数据库使用的是 MySQL。原项目是PC端，我做了个移动端，一些功能在基础上做了修改。

#### 系统环境配置

系统开发平台：JDK1.8 + Windows 10 + Maven3.6.1 

开发语言：JavaEE+vue2.x 

后台框架：Springboot2.x 

前端：Vue2.x

数据库和工具：MySql5.7   Navicat  

开发工具： Intellij Idea, VSCode 

浏览器：Chrome
